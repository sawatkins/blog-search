API Documentation

Blog Search provides a simple API to search through blog content programmatically.

Search Endpoint

GET /api?q={query}

Returns: JSON object with an array of search results, or an empty array if no results are found.

Response Format

{
  "results": [
    {
      "url": "https://example.com/blog-post",
      "title": "Example Blog Post",
      "date": "2023-04-15",
      "snippet": "Post content snippet..."
    },
    // More results...
  ]
}

Example Usage

# Python
r = requests.get("https://blogsearch.io/api", params={"q": "your search query"})
results = r.json()["results"]
